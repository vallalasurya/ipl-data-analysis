# 🏏 IPL Data Analysis Project

## 📊 Overview
This project analyzes IPL cricket data using Python to extract meaningful insights about team performance.

---

## 🔧 Tools & Technologies
- Python
- Pandas
- Matplotlib

---

## 📈 Key Insights
- 🏆 Best team based on win percentage
- 📊 Most consistent team using standard deviation
- 🔥 Highest average scoring team
- ⚡ Performance in high scoring matches (>150 runs)

---

## 📁 Dataset
- IPL match data (CSV file)

---

## 🚀 How to Run
1. Install required libraries:
   pip install pandas matplotlib

2. Run the script:
   python ipl_analysis.py

---

## 📌 Results
- Identified top-performing teams
- Compared team consistency and scoring patterns
